# 🏎️ Koenigsegg Performance Analytics Dashboard

A production-grade, interactive multi-page analytics dashboard built with HTML, CSS, and Chart.js — inspired by Power BI. Designed as a portfolio project showcasing Business Intelligence and Data Visualization skills.

**🔗 Live Demo:** `https://YOUR-USERNAME.github.io/koenigsegg-dashboard`

---

## 📊 Pages & Features

| Page | Content |
|------|---------|
| **Overview** | Production trend (2002–2024), revenue mix donut, HP & price evolution charts |
| **Models** | Full registry table (9 models), units produced, power-to-price ratio |
| **Markets** | Regional sales growth, customer acquisition funnel, buyer demographics |
| **Campaigns** | Brand value growth, digital reach, ROI by channel, media impact |
| **Records** | World record cards, milestone timeline, 0–400–0 chart, speed vs competition |

### ✨ Interactive Features
- Smooth animated page transitions
- Custom gold cursor with hover ring effect
- Era filter (CC / Agera / Modern) updating KPIs dynamically
- Charts load on-demand per page
- Fully responsive layout

---

## 🛠️ Tech Stack

- **HTML5 / CSS3 / Vanilla JavaScript** — zero frameworks, zero dependencies to install
- **Chart.js 4.4.1** — all data visualizations
- **Google Fonts** — Bebas Neue + Barlow
- **GitHub Pages** — free hosting

---

## 🚀 How to Deploy (GitHub Pages)

1. Fork or clone this repository
2. Go to your repo → **Settings** → **Pages**
3. Under **Source**, select `Deploy from a branch`
4. Set branch to `main` and folder to `/ (root)`
5. Click **Save**
6. Your site goes live at: `https://YOUR-USERNAME.github.io/koenigsegg-dashboard`

> ⏱️ It may take 1–2 minutes to go live after saving.
